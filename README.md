# FactWise Employees Dashboard

Frontend assignment submission. Built with React, TypeScript, and AG Grid.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Stack

- React 18 + TypeScript
- Vite
- AG Grid Community
- Recharts
- Tailwind CSS

## Features

- Employee table with sorting, filtering, column resizing, and pagination
- Search by name, email, or location
- Filter by department
- Active employees toggle
- KPI summary cards
- Department headcount and salary charts
- Export filtered data as CSV
